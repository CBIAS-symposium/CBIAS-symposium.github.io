napari Workshop 2024


27-28 November 2024
The Francis Crick Institute, London, UK

Programme in short:

  • Introduction to conda environments
  • Introduction to napari GUI
  • napari plugins showcase
  • Basic image analysis in Jupyter notebook and napari
  • Work on your own data projects


    • The course is open to max 25 trainees and will focus on the use of napari for bioimage analysis projects proposed by the participants.
    Target audience: students and researchers with some coding experience, whether python or ImageJ macro only.
    Learning outcomes: be able to install napari with different command line methods; understand the commands in the software graphical user interface; know how to install plugins to add new functionalities; understand the basic of software API and use python code to process and render image data in napari. Learn how to install jupyter notebook and useful python packages for basic image analysis. Learn the basic methods to analyse image data sets with napari GUI, plugins, or python packages.
